# Golda Coffee - Scrollytelling Landing Page

A high-end, immersive landing page for a coffee brand "Golda".
Built to demonstrate advanced scrollytelling techniques, smooth animations, and premium UI design.

**[View Demo](https://goldacoffee.vercel.app)**

## Features

- **Cinematic Scrollytelling**: Image sequence powered by HTML5 Canvas and scroll synchronization.
- **Smooth Momentum Scrolling**: Integrated `Lenis` for a buttery smooth experience.
- **Advanced Animations**: Powered by `motion` (formerly framer-motion) v12.
  - Staggered reveals
  - Split-text scroll effects
  - Count-up statistics
  - Full-screen animated navigation
- **Responsive Design**: Mobile-first approach with custom layouts for phone and desktop (e.g., Bento Grid adaptation).
- **Performance**: Optimized asset loading and canvas rendering.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Motion](https://motion.dev/)
- **Fonts**: [Outfit](https://fonts.google.com/specimen/Outfit) (Google Fonts)

## Getting Started

1. **Clone the repository**:

   ```bash
   git clone https://github.com/ifalfahri/golda.git
   cd golda
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Run the development server**:

   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
